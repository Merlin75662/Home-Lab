# 🏠 My Home Lab
This is my personal learning environment where I practice Network Administration and Cybersecurity.

## 🖥️ The Hardware
* **Server:** Dell PowerEdge R630.
* **Graphics Card:** NVIDIA Tesla P4 (Used for local AI projects).
* **3D Printers:** Ender 3 and Ender 3 S1 (Running Klipper firmware).

## 🌐 What I’m Running
* **Proxmox:** This is my "Hypervisor." It lets me run many different virtual computers on my one R630 server.
* **Pi-hole:** A DNS sinkhole that blocks ads for every device in my home.
* **Nginx Proxy Manager:** Helps me manage my web traffic and keep my services secure.
* **Ollama & Open WebUI:** My private, local AI setup powered by the Tesla P4.
* **Game Servers:** I host private servers for World of Warcraft (AzerothCore) and Palworld.
