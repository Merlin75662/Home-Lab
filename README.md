# 🏠 My Home Lab
This is where I learn Network Administration and Cybersecurity.

## 🖥️ The Server
* **Model:** Dell PowerEdge R630
* **Brain (OS):** Proxmox (This lets me run many "virtual" computers at once)
* **Graphics Card:** NVIDIA Tesla P4 (Used for my AI projects)

## 🌐 What it Does
* **Pi-hole:** Blocks ads for every device in my house.
* **Nginx Proxy Manager:** Helps me access my projects safely.
* **Ollama:** Runs my own private AI (like ChatGPT, but local).
* **Game Servers:** I host World of Warcraft and Palworld for fun.

## 🛠️ Other Tools
* **3D Printing:** I use an Ender 3 and Ender 3 S1 to build parts for my server.
* **Certs:** I have my CompTIA Network+ and Tech+.
